# TOOLS.md - 本地笔记

Skills 定义工具_如何_工作。这个文件用于_你的_具体信息——也就是你配置里独有的内容。

## 这里写什么

例如：

- 摄像头名称和位置
- SSH 主机和别名
- 偏好的 TTS 声音
- 音箱/房间名称
- 设备昵称
- 任何与环境相关的内容

## 为什么要分开？

Skills 是共享的。你的配置属于你自己。把它们分开意味着你可以在不丢失笔记的情况下更新 skills，也能在不泄露你的基础设施信息的情况下共享 skills。
